| 支持格式        | 描述                             |
| ----------- | ------------------------------ |
| .bmp        | Windows 画图软件默认保存的格式，位图         |
| .jpeg（.jpg） | 有损压缩图片，通常用于照片显示                |
| .png        | 便携式网络图像，用于 logo，背景图形等。支持透明和半透明 |
| .gif        | 动画 ，如：表情包                      |
| .svg        | 矢量图片                           |
| .webp       | 最新的压缩算法，非常优秀的图片格式              |
