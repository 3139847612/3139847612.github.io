从当前网页出发，要找到图片的路径就叫 相对路径
```
<img src="images/logo.png" />
<img src="../images/logo.png" />
<img src="../../images/logo.png" />
.....
```
`../` 表示回退上一级目录
`./` 表示当前目录
通过 `cmd`命名行 `dir`命令可查

绝对路径
描述文件或文件夹的精准完整地址
```
<img src="D:\web\icoding-web\images\logo.png" />
<img
  src="https://www.icodingedu.com/files/system/2019/09-25/22132557f330499313.png"
/>
```
