
| 标签        | 描述                         |
| --------- | -------------------------- |
| `<table>`   | 定义表格                       |
| `<th>`      | 定义表格的表头                    |
| `<tr>`      | 定义表格的行                     |
| `<td>`      | 定义表格单元                     |
| `<caption>` | 定义表格标题（作为 table 的第一个子元素出现） |
| `<thead>`   | 定义表格的页眉                    |
| `<tbody>`   | 定义表格的主体                    |
| `<tfoot>`   | 定义表格的页脚                    |
