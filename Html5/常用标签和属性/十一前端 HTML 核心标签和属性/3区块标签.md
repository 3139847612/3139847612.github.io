html5 新增的语义化标签

| 标签名称    | 描述                |
| ------- | ----------------- |
| header  | 定义页头              |
| nav     | 定义导航              |
| main    | 定义页面的主体区域         |
| aside   | 可用作文章的侧栏          |
| article | 可用作文章的内容          |
| section | 可用作文档的区域块，类似于 div |
| footer  | 定义页脚              |
