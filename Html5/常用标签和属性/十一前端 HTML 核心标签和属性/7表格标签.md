
| 标签名     | 描述                     |
| ------- | ---------------------- |
| table   | 表格标签                   |
| tr      | 表格行                    |
| td      | 表格列                    |
| th      | 标签，可替代 td 标签，用来设置表格的标题 |
| thead   | 定义表格头部                 |
| tbody   | 定义表格主体内容               |
| tfoot   | 定义表格尾部                 |
| caption | 设置表格的标题                |
