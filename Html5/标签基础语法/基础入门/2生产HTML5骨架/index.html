    生产HTML5骨架

    输入 ! 英文模式下，按 tab键，即可自动生成 HTML5 的骨架 注：如果没有生成，看网页的格式是否正确或保存 在 Vscode 中 生成 HTML5 骨架 即可

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>这是我的第一个HTML网页</title>
  </head>
  <body>
    第一个HTML网页 ，Hello ！
  </body>
</html>
