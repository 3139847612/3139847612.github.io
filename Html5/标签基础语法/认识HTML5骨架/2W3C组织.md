当前HTML5 已经一统江湖了，早期的 HTML4 及以前的版本都已不用，只做 HTML 发展历史了解就好。这些版本的制定者为W3C 组织
W3C 指万维网联盟（World Wide Web Consortium）是万维网的主要国际标准组织
W3C 创建于 1994 年 10 月主要负责制定 WEB 标准
W3C 由 _Tim Berners-Lee_（蒂姆·伯纳斯·李） 创建 ，被誉为 "互联网之父"
W3C 是一个会员组织
W3C 的工作是_对 web 进行标准化
W3C 创建并维护 _WWW 标准_
W3C 标准也被称为 _W3C 规范_
主要是 HTML 和 CSS
JavaScript 由 Brendan Eich 于 1995 年发明，并于 1997 年成为 ECMA 标准
